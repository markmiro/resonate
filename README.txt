* Make sure ruby is installed with all the required gems.
    - I believe the lowest required version is 1.9, but it works with 2.0 for sure
    - do "gem list" in terminal, verify that "bundle" is installed
    - then do "bundle install"
* do "rackup" to run the server
* do "rackup -p 3000" or whatever number to run in certain port
* Open the browser with: "open /Applications/Google\ Chrome\ Canary.app --args --disable-web-security"
* Type "http://localhost:3000/" or whatever number into the location bar to run
